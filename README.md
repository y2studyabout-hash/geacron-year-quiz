# GeaCron Year Quiz Helper

GeaCron の年号表示を一時的に隠し、  
「年代を推測して当てるクイズ形式」で学習できる Tampermonkey ユーザースクリプトです。

- 古代 / 中世 / 近代 / 現代 からランダム出題
- 年号入力 → 答え合わせ → 正誤判定
- 判定メッセージはふわっと表示 → 5 秒後にフェードアウト
- 直前結果を「地図更新ボタンの右」に履歴表示
- UI は地図コントローラ左側に固定（画面を圧迫しません）

> 個人利用向けのブラウザ UI カスタマイズスクリプトです。


---

## 🔧 対応環境

- Chrome / Edge / Brave / Vivaldi など  
- Tampermonkey が利用できるブラウザ

対象サイト：

https://www.geacron.com/

---

## 🧩 インストール方法

1. ブラウザに Tampermonkey をインストール
2. 次の URL を開く（Raw 形式のスクリプト）  
   👉 **<https://raw.githubusercontent.com/y2studyabout-hash/geacron-year-quiz/main/geacron-year-quiz.user.js>**
3. Tampermonkey のインストール画面で「Install」を選択
4. GeaCron を開き直すと UI が表示されます


---

## 🎮 使い方

- **古代R / 中世R / 近代R / 現代R**
  → それぞれの時代からランダム出題
- 入力欄に推測年を入れて **答え合わせ**
- 許容誤差は時代に応じて変化  
  - 古代：±50 年  
  - 中世：±20 年  
  - 近代：±10 年  
  - 現代：±5 年
- 直前の結果は「更新ボタンの右」に履歴表示
- 年号マスク解除・直前の年へ戻す機能あり


---

## 🛠 トラブルシューティング

- UI が表示されない  
  → ページを再読み込み  
- 競合が起きた  
  → Tampermonkey からスクリプトを一時停止  
- 年号が更新されない  
  → ページを再読み込みして再試行
